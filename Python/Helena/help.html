Ajuda sobre o Helena

Software para projetos de Transformadores

O aplicativo Helena é um software para cálculos de transformadores de várias potências e tamanho.

Ele é um software de código fonte aberto e livre com o propósito de trocas de melhorias e evolução em uma área pouco explorado no universo de software livre, como o LibreCAD (http://www.librecad.org), o FreeCAD (http://www.freecad.org) entre outros.

Neste manual de ajuda, encontram-se as funções de cada item do programa como também sua equação relacionada.

Para mais informações, acesse http://www.everton.mat.br/helena
