This is a software to create mono phase transformer project.

This informations are in English or Portuguese

O transformador pode ser visto conforme a Figura 1 (a) obtida em https://en.wikipedia.org/wiki/Transformer, exibe informações técnicas sobre transformador.

O programa é utilizado para automatizar os cálculos de transformadores de até 3 kVA monofásicos, com no máximo três condutores na entrada e saída.

Alguns campos já vem setado, sendo os campos em azul, para que facilite, necessitando preencher os valores de tensão de entrada e saída, e potência de saída.

Ainda está em teste a criação da folha de fabricação com as informações relevantes na produção.

O pontos deste projeto são:

  • Ponto 1 - Ser um software live para criação de projetos de transformadores;
  • Ponto 2 - Ser de uso livro para estudo quanto comercial;
  • Ponto 3 - Uso de ferramentas livres e integrações com outras plataformas;
  • Ponto 4 - Fazê-lo crescer, e conquistar o mundo.
  • Ponto 5 - Que ele seja utíl para você :)
